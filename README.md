# 🌍 TS Academy Capstone Project – Planetary Data Landing Page

This is a responsive and visually engaging landing page built to present planetary data in a clean, structured, and interactive way. This project is an individual submission for the TS Academy Capstone program.

## ✨ Features

- 📱 Fully responsive (desktop, tablet, mobile)  
- 🎨 Clean and modern UI design  
- 🧭 Structured layout (Hero, Data, Visuals, Table, Form)  
- 🧩 Component-based architecture (React + Vite)  
- 🎥 Video integration  
- 🖼️ Interactive image grid (hover + mobile tap support)  
- 📝 Form with validation and success modal  
- 💎 Premium input styling and micro-interactions  
- 🧭 Smooth UI interactions (hover effects, animations)  

---

## 🛠️ Tech Stack

- React (Vite)  
- JavaScript (ES6+)  
- CSS / Responsive design  

---

## 📁 Project Structure

- `src/components/` – Reusable React components  
- `src/assets/` – Images, videos, and other media  
- Clean separation of sections (Hero, Data, Visuals, Form, Footer)
